# Copilot Project Submission

## Overview

This submission demonstrates the use of Microsoft Copilot to complete a series of supply chain management tasks. The project includes email summarization, meeting planning, report generation, data analysis, and presentation creation — all assisted by Copilot.

## Included Files

| File | Description |
|------|-------------|
| `copilot prompts.docx` | Documentation of all Copilot prompts used throughout the project |
| `task outputs/1.1 Email summary.docx` | Summarized email content related to supply chain communications |
| `task outputs/1.2 Supply Chain Report.docx` | Report synthesizing supply chain information from emails |
| `task outputs/2.1 Supply Chain Issues Review.ics` | Calendar invite for a supply chain issues review meeting |
| `task outputs/2.2 Meeting Doc and Action plan.docx` | Meeting document with agenda and action plan |
| `task outputs/3.1 Assessment Supply Chain Metrics.xlsx` | Spreadsheet with supply chain metrics and assessment data |
| `task outputs/3.2 Supply Chain Issue Analysis Report.docx` | Analytical report on supply chain issues |
| `task outputs/Supply Chain Issue Compliance Report.pptx` | Presentation summarizing compliance findings |

## Task Overview

### Task 1 — Email Summarization & Reporting

- **1.1 Email Summary**: Uses Copilot to summarize key supply chain emails, extracting critical information and action items.
- **1.2 Supply Chain Report**: A written report consolidating insights from email communications into a structured supply chain overview.

### Task 2 — Meeting Planning & Action Items

- **2.1 Supply Chain Issues Review (Calendar Invite)**: A meeting invite (.ics) scheduled to review ongoing supply chain issues with relevant stakeholders.
- **2.2 Meeting Doc and Action Plan**: A detailed meeting document including the agenda, discussion points, and a follow-up action plan.

### Task 3 — Data Analysis

- **3.1 Assessment Supply Chain Metrics**: An Excel spreadsheet containing supply chain metrics used for assessment and analysis.
- **3.2 Supply Chain Issue Analysis Report**: A written report analyzing supply chain issues based on the metrics data.

### Task 4 — Stakeholder Report Creation & Presentation

- **Supply Chain Issue Compliance Report (PPTX)**: A PowerPoint presentation generated using Copilot in PowerPoint, serving as the stakeholder compliance report. The presentation:
  - Accurately reflects information from the "Supply Chain Issue Analysis Report.docx" and "Supply Chain Report.xlsx"
  - Features a professional, consistently formatted design
  - Includes speaker notes for presenter guidance

## How to Navigate and Review

1. **Start with `copilot prompts.docx`** — Review the prompts used to understand how Copilot was leveraged for each task.
2. **Follow the task numbering** — Files are prefixed with their task number (1.1, 1.2, 2.1, etc.) for easy sequential review.
3. **Review by task group**:
   - Tasks 1.x → Email and reporting deliverables
   - Tasks 2.x → Meeting planning deliverables
   - Tasks 3.x → Data analysis deliverables
   - Task 4 → Stakeholder presentation deliverable
4. **All output files are in the `task outputs/` folder** — Each deliverable corresponds to a specific task requirement.
